## About the Project

Okan Kaplan Academy is an educational initiative designed to provide practical and applied learning in the field of technology and engineering.

The platform focuses on real-world implementations rather than theoretical explanations.

Main learning areas include:

- Internet of Things (IoT)
- Embedded Systems Development
- Arduino and Raspberry Pi programming
- Sensor integration and hardware control
- Smart device development
- Practical engineering projects

---

## Technology Stack

The project is built using modern and lightweight web technologies:

- HTML5 (semantic structure)
- CSS3 (responsive design)
- JavaScript (client-side interactions)
- Bootstrap 5 (UI framework)
- Font Awesome (icon library)
- GitHub Pages (hosting platform)
- JSON-LD structured data (SEO optimization)

- Fast and lightweight static website architecture
- Fully responsive design for all devices

- Open Graph and Twitter Card support
